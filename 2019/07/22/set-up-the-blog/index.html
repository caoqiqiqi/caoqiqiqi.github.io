  1. 安装nodejs和git环境
    node –verion
    npm –version
    git –version
  2. 使用nodejs的包管理工具npm安装hexo
    npm install -g hexo-cli
  3. 创建博客目录
    hexo init blog
  4. 进入目录
    cd blog
  5. 安装依赖项
    npm install
  6. 添加自己的博客文章（markdown格式）
    markdown文件拷贝到blog目录下的source/_posts
    hexo g
  7. 启动服务器
    hexo s -p 80
    http://<服务器公网IP地址>

django学习

django

虚拟环境

环境的存在意义

实现方式

1.pycharm自己创建

2.python3中可使用自带venv模块进行创建

python -m venv 环境名

3.virtualenv可用于python2和3中

virtualenv –no-site-packages -p 环境名

4.使用虚拟技术(docker)

django项目

安装django: pip install django==2.0.7

创建项目: django-admin startproject 项目名

init: 初始化, pymysql.install_as_mysqldb()

settings.py: 配置, 数据库配置, app配置

urls.py: 路由文件, 定义路由和方法

创建应用: python manage.py startapp 应用名

init.py

views.py: 视图文件(写代码逻辑)

models.py: 模型文件, 定义模型

admin.py: 后台文件,定义注册到django后台

迁移

1.迁移django自带的表

python manage.py migrate

2.迁移自定义的模型

1.生成迁移文件 python manage.py makemigrations
2.执行迁移中间文件 python manage.py migrate

后台

创建账号: python manage.py createapp 项目名

将博客托管到三方平台 - github

  1. 创建一个名为<用户名>.github.io的项目（仓库）
  2. 修改hexo的配置文件_config.yml指定托管到github
    deploy:
    type: git
    repo:
    branch: master
  3. 安装一键部署到github的插件
    npm install hexo-deployer-git –save
  4. 在source文件下,touch CNAEM , 然后再CNAME中加入域名(caoqi.top)
  5. 部署
    hexo d -g / hexo g -d
  6. 配置CNAME域名解析
    将自己的域名解析到github.io的域名
